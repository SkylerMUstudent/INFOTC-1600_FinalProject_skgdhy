# How to create Scalable Vector Graphics
### Written by Skyler Goodale
This guide will help you create **Scalable Vector Graphics (SVG)**. This guide will cover the basics of how to create different graphics such as shapes, colors, paths, and even text.

The target Audience for this tutorial is first time users and beginners to early web design and graphics design. It more focused on the basics of how to build a SVG and some creative different things you can do with it.

- [Information about SVG](Info_about_SVG.md) : This page explains what Scalable Vector Graphics is.
- [How to create a SVG](basics_howTo.md) : This page will go with a step by step guide on how to create a SVG.
